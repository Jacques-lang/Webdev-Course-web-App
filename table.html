Commonly Used Html Tags

Html Tags Description
<p> This tag defines a paragraph.
<html> This tag is the root of an HTML document. All other tags are nested within this tag.
<head> This tag contains meta-information about the document, such as its title and links to stylesheets.
<title> This tag sets the title of the document, which appears in the browser tab.
<body> This tag contains the content of the document, such as text, images, and other elements.
<h1> to <h6> These tags define headings of different sizes, with <h1> being the largest and <h6> being the smallest.
<a> This tag creates a hyperlink to another web page or resource.
<img> This tag embeds an image in the document.
<ul> This tag creates an unordered list, where each item is marked with a bullet point.
<ol> This tag creates an ordered list, where each item is numbered.
<li> This tag defines an item in a list, either ordered or unordered.
<table> This tag creates a table to display data in rows and columns.
<tr> This tag defines a row in a table.
<th> This tag defines a header cell in a table.
<td> This tag defines a data cell in a table.
<div> This tag defines a division or section in an HTML document.
<span> This tag defines a section in an HTML document that can be styled separately from the rest of the content.
<form> This tag creates a form for user input, such as text fields and buttons.
<input> This tag creates an input field in a form, such as a text box or a radio button.
<button> This tag creates a clickable button in a form.
<label> This tag defines a label for an input field in a form.
